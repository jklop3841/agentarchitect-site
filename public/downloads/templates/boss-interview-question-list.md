# 老板访谈问题清单

作者：智能体架构师卢成 / Jack Lu  
主站：https://agentarchitect.me

## 目标

把老板口中的“我想做 AI / 我想做智能体 / 我想做短视频矩阵”翻译成真实业务问题。

## 1. 业务问题

1. 你现在最想解决的业务问题是什么？
2. 这个问题持续多久了？
3. 如果不解决，会损失什么？
4. 如果解决，最直接的收益是什么？

## 2. 成本问题

1. 现在这个流程每月消耗多少人力？
2. 是否有重复劳动？
3. 是否有员工不愿意做、做不好、容易出错的环节？
4. 哪些地方最浪费老板时间？

## 3. 增长问题

1. 现在客户从哪里来？
2. 线索来了以后谁跟？
3. 哪个环节最容易丢客户？
4. 你最希望 AI 帮你提升哪个转化环节？

## 4. 管理问题

1. 现在是否有标准流程？
2. 员工是否按流程执行？
3. 老板是否能看到数据？
4. 哪些事情最依赖老板本人？

## 5. 风险问题

1. 有没有客户隐私？
2. 有没有高风险承诺？
3. AI 出错谁兜底？
4. 哪些事情绝不能自动化？

## 6. 预算与决策

1. 你愿意先做小试点还是一步到位？
2. 你希望多久看到效果？
3. 谁来验收？
4. 是否愿意提供真实业务数据？
